Maze Escape Simulation - CENG 202 Project

Instructions:
-------------
1. Compile all .java files:
   javac *.java

2. Run the simulation:
   java Main

3. When prompted, enter:
   - Maze width and height (e.g., 6, 8)
   - Number of agents (e.g., 3)
   - Max Turns

4. The simulation will begin.

5. Log output is saved to 'simulation_log.txt'
